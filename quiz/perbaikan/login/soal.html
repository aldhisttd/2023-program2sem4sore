LOGIN (Perbaikan)

Pilihan Soal

  1. Seri soal ada 2, GENAP dan GANJIL
  2. Silahkan pilih soal sesuai dengan angka digit terakhir NIM kalian, Ganjil atau Genap
  3. Untuk angka digit akhir 0, pilih yang GENAP

Instruksi

  1. Source branch = main
  2. Ketentuan nama branch : perbaikan/login_NIM
  3. Jika sudah selesai lakukan PR dengan judul PR : perbaikan/login/NIM/NAMA_LENGKAP
  4. Mention saya di bagian description PR kalian
  5. Add saya sebagai Reviewer di PR kalian

Soal

  1. Lokasi project berada di folder quiz/perbaikan/login/NIM
  2. Buat sistem login dengan multi user dan multi role dengan ketentuan table dibawah
  3. Lengkapi semua validasi nya
