Pilihan Soal

Instruksi Soal

  1. Buat dulu branch anda dari branch master
  2. Ketentuan nama branch adalah: quiz03/login_NIM
  3. Lokasi project kalian beradan di folder quiz/03_login/NIM
  4. Buat login dengan 3 role dan 3 akun login, "admin", "user", "karyawan"
  5. Jika login sbg admin, maka redirect ke halaman admin jika user ke halaman user, jika karyawan ke halaman karyawan
  6. Lakukan validasi halaman, contoh login admin tidak boleh akses halaman user lain dan begitu pula user lainnya
  7. Lakukan PR dengan nama branch tadi dan jangan lupa untuk melampirkan hasil output dari program
  8. Jangan lupa di PR nya untuk menambahkan saya sebagai reviewer, ada di bagian kanan ketika PR
